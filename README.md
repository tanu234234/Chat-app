# Chat Application

## Overview

This project is a **Real-Time Chat Application** built using **Node.js, Express.js, and Socket.IO**. It enables multiple users to communicate instantly through a web interface. Messages sent by one user are broadcast to all connected users in real time, creating a seamless live chat experience.

The application demonstrates how modern web technologies enable real-time communication between clients and servers. It uses WebSocket technology through Socket.IO to allow instant data exchange without refreshing the page. Users can join the chat by entering their username and then send messages that are instantly visible to all other connected users.

The project integrates both frontend and backend components. The frontend is responsible for displaying the chat interface and collecting user input, while the backend manages the server logic, handles user connections, and distributes messages to all connected users. This architecture helps in understanding how full-stack applications operate and how real-time systems are implemented in web development.

## Features

* Real-time messaging using **Socket.IO**
* Multiple users can communicate simultaneously
* Username-based messages for easy identification
* Instant message broadcasting to all connected users
* Simple and responsive chat interface
* Dynamic message updates without page refresh
* Frontend and backend integration

## Technologies Used

* **Node.js** – Server-side runtime environment
* **Express.js** – Web framework used to create the server
* **Socket.IO** – Library used for real-time WebSocket communication
* **HTML** – Structure of the chat interface
* **CSS** – Styling and layout of the application
* **JavaScript** – Client-side logic and interaction handling

## Installation and Setup

1. Clone the repository

git clone https://github.com/tanu234234/CHAT APPLICATION.git

2. Navigate to the project folder

cd "CHAT APPLICATION"

3. Install dependencies

npm install

4. Start the server

node server.js

5. Open the application in your browser
   
http://localhost:3000

## Usage

1. Enter your username in the input field.
2. Type a message in the chat box.
3. Click the **Send** button to send the message.
4. The message will appear instantly for all connected users.

You can open the application in multiple browser tabs or different devices to simulate multiple users chatting simultaneously.

## Conclusion

This project demonstrates how **WebSocket-based communication using Socket.IO** enables real-time interaction between multiple users. By integrating frontend technologies with a Node.js backend server, the application provides a practical example of a live chat system where users can exchange messages instantly without refreshing the page.
